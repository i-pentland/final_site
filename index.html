2025 Regular Season ESPN public API

Compare real players using real season stats.

This version focuses on one clear feature instead of fake scouting language: pull player information and 2025 regular season stats from ESPN's public JSON endpoints, choose any two players, and highlight the better number in green.

Comparison rule

Green means the player with the higher value wins that stat. If both values are the same, neither side is highlighted.
