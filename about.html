Project Idea

Matchup Room is

Instead of using made-up numbers, the site pulls actual player information and stats so the comparison feels more grounded in real football data.

How it's built

This project was built with AI-assisted development. Specifically, I used Codex 5.4 for debugging, brainstorming, refining parts of the JavaScript logic, and helping locate APIs. AI was part of roughly 50 to 75 percent of the process, but the core idea, user interface, and overall structure of the site were my own. I chose the football comparison concept, designed the interface, and laid out the simpler page structure and JavaScript interactions myself. Then, I had AI mainly to speed up iteration and problem-solving.

Data Source

The project uses nflverse player metadata and 2025 regular season player stats. The page first tries the live public files, and if that fails it uses the local copies bundled with the project.

That setup helps the site keep working more reliably on GitHub Pages or a local server.

Why I Built It

I wanted the final project to be more specific than a generic sports page. The comparison feature and team dashboard make it feel closer to a small football analytics tool instead of just a list of stats.
