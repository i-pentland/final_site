Project Idea

Part of me has always been absolutely convinced I could be a better General Manager for the Giants than Joe Schoen. Many decisions he make look bad on paper, and turn out to be worse in real life.

I created this site so that, during the 26-27 NFL season, I can have a deeper insight into the trades that these managers are making.

How it's built

This project was built with AI-assisted development. Specifically, I used Codex 5.4 for debugging, brainstorming, refining parts of the JavaScript logic, and helping locate APIs. AI was part of roughly 75 percent of the process, but the core idea, user experience, and overall structure of the site were my own. I had AI mainly to speed up iteration, complex JS, and problem-solving.

Data Source

This project pulls live ESPN JSON APIs (team rosters, player stats/overview, team statistics) instead of using embedded data. When I started the project, I was using a CSV file from nflverse on GitHub to get the logic right before tinkering with actual API calls.

That setup helps the site keep working more reliably on GitHub Pages or a local server.
